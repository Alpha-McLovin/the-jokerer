@Composable()
fun AppStructure(navigationType: NavigationTypes, navController: NavHostController = rememberNavController())
Content copied to clipboard

This composable represents the overall structure of the application.

The AppStructure composable sets up the application layout based on the specified navigationType. It includes top and bottom bars, navigation components, and hosts the content for different destinations.

Parameters

navigationType

The type of navigation to be used, either NavigationTypes.BOTTOM_NAVIGATION or NavigationTypes.NAVIGATION_RAIL for portrait or landscape.

navController

The navigation controller to handle navigation between different destinations.
